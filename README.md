# STEMS - Smart-Task-Evaluation-Management-System

## Overview

STEMS (Smart Task Evaluation Management System) is a web-based application designed to streamline the assignment, tracking, evaluation, and management of tasks within an organization or educational institution. The platform provides a centralized environment where users can create, manage, monitor, and evaluate tasks efficiently while maintaining transparency and accountability.

The system aims to reduce manual effort, improve task tracking, enhance productivity, and provide structured task evaluation through an intuitive and user-friendly interface.

---

## Features

### User Module

* User registration and login
* View assigned tasks
* Submit task updates and progress
* Track task status
* View evaluation results and feedback
* Receive notifications and announcements
* Monitor personal performance metrics

### Administrator Module

* Create and assign tasks
* Manage users and teams
* Monitor task progress
* Evaluate submitted tasks
* Generate performance reports
* Publish announcements and notifications
* Track overall system activity

### General Features

* Secure authentication system
* User-friendly interface
* Task scheduling and management
* Progress tracking and evaluation
* Real-time updates and notifications
* Performance monitoring and reporting

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/STEMS.git
cd STEMS
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Database

1. Create a MySQL database.
2. Import the provided SQL file.
3. Update database credentials in the configuration file.

### Run Application

```bash
python app.py
```

Open:

```text
http://localhost:5000
```
